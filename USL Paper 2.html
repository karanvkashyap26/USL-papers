Clustering and Dimension reduction using Topic Modeling- Latent Dirichlet Allocation (LDA) using Gensim!

Introduction

Extract topics from Unstructured data!

What is Topic Modeling, and how does it work?

Automatically extracting the topics people are discussing from vast volumes of text is one of the key applications of natural language processing.

Large text examples include social media feeds, consumer reviews of hotels, movies, and other companies, user input, news reports, and e-mails from customers with complaints.

Manually reading through such vast volumes and assembling the topics is extremely difficult.

As a result, we needed an automatic algorithm like LDA to read the text and output the discussed topics.

Topic modeling Use Cases:

Document categorization Document summarization Dimension reduction- Doc words matrix is reduced to doc topic matrix

stop_words = stopwords.words('english')
def remove_stopwords(texts):
    return [[word for word in simple_preprocess(str(doc)) if word not in stop_words] for doc in texts]

def lemmatization(texts, allowed_postags=['NOUN']):
    """https://spacy.io/api/annotation"""
    texts_out = []
    for sent in texts:
        doc = nlp(" ".join(sent)) 
        texts_out.append([token.lemma_ for token in doc if token.pos_ in allowed_postags])
    return texts_out

How LDA functions is as follows:

The dictionary and the corpus are the two key inputs to the LDA topic model.

Dictionary : Specific identifiers for each unique word For eg, tv' is 0, future' is 1, launched' is 2, pause' is 3, multimedia' is 4, and chief' is 5.

Corpus: The number of times a specific word appeared in each text is known as the corpus (Document Term Matrix) [(0, 1), (1, 2), (2, 1), (3, 1), (4, 1)]

For example, (0, 1) indicates that word id 0 (word: tv') appears only once in the first text. In the same way (1,2), the word id 1 (word: add') appears twice, and so on.

It transforms a corpus of Document Term Matrix into two other matrices: document-topic matrix and topic-term matrix, given a corpus of Document Term Matrix.

The ultimate aim of LDA is to find the best representation of the doc-topic and topic-term matrices.

Gensim creates a unique id for each word in the document. The produced corpus shown above is a mapping of (word_id, word_frequency.

All we need to train the LDA model is present here. We must also include the number of topics(k') in addition to the corpus and dictionary. We can pick k' based on our needs.

K: Number of topics

N: Number of words in the document

M: Number of documents to analyze

: Per-document topic distribution

High means every document is likely to contain a mixture of most of the topics and not just any single topic specifically.

Low means A document is more likely to be represented by just few of the topic.

: Per topic word distribution

High means each topic is likely to contain a mixture of most of the words not just any word specifically.

Low means topic may contain a mixture of just a few of words.

(k): Word distribution for topic k (Topic probability per word)

(i): Topic distribution for for document I (Topic probability per document)

Z(i,j): Topic assignment for w(i,j)

w(i,j): j-th word in i-th document

Aside from that, alpha and eta are hyperparameters that control the topic sparsity. The number of documents to be included in each training chunk is referred to as chunk size. Controls the amount of documents that are processed at one time.

def format_topics_sentences(ldamodel=None, corpus=corpus, texts=data):
    # Init output
    sent_topics_df = pd.DataFrame()

    # Get main topic in each document
    for i, row in enumerate(ldamodel[corpus]):
        row = sorted(row[0], key=lambda x: (x[1]), reverse=True)
        # Get the Dominant topic, Perc Contribution and Keywords for each document
        for j, (topic_num, prop_topic) in enumerate(row):
            if j == 0:  # => dominant topic
                wp = ldamodel.show_topic(topic_num)
                topic_keywords = ", ".join([word for word, prop in wp])
                sent_topics_df = sent_topics_df.append(pd.Series([int(topic_num), round(prop_topic,4), topic_keywords]), ignore_index=True)
            else:
                break
    sent_topics_df.columns = ['Dominant_Topic', 'Perc_Contribution', 'Topic_Keywords']

    # Add original text to the end of the output
    contents = pd.Series(texts)
    sent_topics_df = pd.concat([sent_topics_df, contents], axis=1)
    return(sent_topics_df)
